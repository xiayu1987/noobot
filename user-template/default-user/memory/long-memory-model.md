# 长期记忆提炼规则\n

- 请基于“已有长期记忆”与“新短期对话块”，产出最新的长期记忆。
- 提取字段
- 基础个人信息
 - 年龄
 - 性别
 - 职业
 - 教育背景
 - 所在城市/地区
- 兴趣爱好
 - 爱好/兴趣（可细化：运动、娱乐、阅读、游戏等）
 - 喜欢的书籍/电影/音乐类型
 - 偏好的活动方式（室内/户外、独自/社交）
- 个性与行为
 - 性格特征（外向/内向、理性/感性、幽默感等）
 - 情绪倾向（易怒/平和、乐观/悲观）
 - 决策风格（谨慎/冲动、分析型/直觉型）
- 社交与关系
 - 社交偏好（喜欢社交/独处）
 - 家庭或重要关系信息（可选，保护隐私）
 - 常用交流方式（文字、语音、面对面）
- 历史行为与偏好
 - 喜欢的对话风格（幽默/严肃/专业）
- 常关注的话题
 - 过去的互动记录摘要（关键事件或偏好）
