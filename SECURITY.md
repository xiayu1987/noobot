# Security Policy

[中文](./SECURITY.zh-CN.md) | English

If you discover a security vulnerability, **please do not open a public issue**.

Please report it privately to the maintainer with:

- Vulnerability description
- Reproduction steps
- Impact scope
- Suggested fix (optional)
